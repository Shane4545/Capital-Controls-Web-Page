CCI hero loop — lineage
========================

File: cci-hero-loop.mp4 (homepage cinematic hero, muted, looped in JS)

Preferred build (AI image-to-video from your own field photo — no stock footage):

  set REPLICATE_API_TOKEN=your_token
  pip install requests pillow imageio-ffmpeg
  python scripts/build_hero_loop_ai.py

Uses Replicate model christophy/stable-video-diffusion (Stable Video Diffusion) via HTTPS
(no `replicate` Python package — works on Python 3.14+).
Input image default: assets/photos/wtp-rockland-actiflo-panel.jpg

Fallback without API token (non-AI Ken Burns from the same still):

  pip install pillow imageio imageio-ffmpeg numpy
  python scripts/build_hero_loop_from_still.py

After replacing the MP4, bump the cache-bust query in cinematic-home.js
(CCI_HERO_MP4 … ?v=N) so browsers fetch the new file.
